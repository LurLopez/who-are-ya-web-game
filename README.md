# Clon de "Who Are Ya" - Proyecto de Sistemas Web

Este repositorio contiene el desarrollo de una aplicación web inspirada en el juego "Who Are Ya", realizada como proyecto grupal para la asignatura de **Sistemas Web** en la Facultad de Informática de la **UPV/EHU**.

## 🚀 Descripción del Proyecto
El objetivo del proyecto fue diseñar y desarrollar una plataforma web funcional que permitiera a los usuarios jugar a adivinar futbolistas mediante pistas, integrando una arquitectura completa de cliente-servidor.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML,EJS, CSS y JavaScript.
* **Backend:** Node.js
* **Base de Datos:** MongoDB
* **Protocolos:** Comunicación asíncrona mediante Fetch API.

## 👤 Mi Contribución
Al ser un proyecto de equipo, me encargué específicamente de:
* La **lógica de conexión** entre el Front-end y el Back-end.
* El diseño y la gestión de la **base de datos**, asegurando la persistencia de la información.
* Implementación de la arquitectura **cliente-servidor**.

## 📈 Aprendizajes
Este proyecto me permitió profundizar en el desarrollo Full-stack, la gestión de estados en el navegador y la importancia de una arquitectura bien estructurada para el intercambio de datos en tiempo real.
